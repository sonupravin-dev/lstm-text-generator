# LSTM Text Generation using Generative AI

## 📌 Overview
This project implements a text generation model using LSTM (Long Short-Term Memory). The model learns patterns in text data and generates new text based on a seed input.

## 🎯 Objective
To build a deep learning model that can predict the next word in a sequence and generate meaningful sentences.

## 📂 Dataset
A small custom dataset is used:
once upon a time there was a king  
the king had a queen  
the queen loved the king  
the kingdom was peaceful  

## ⚙️ Methodology
1. Text preprocessing (lowercase, tokenization)
2. Sequence creation
3. Padding sequences
4. LSTM model training
5. Text generation

## 🧠 Model Architecture
- Embedding Layer
- LSTM Layer
- Dense Layer (Softmax)

## 📊 Output
Example:
once upon a time there was a king  
the king had a queen who loved the king  

## 🛠️ Tools Used
- Python
- TensorFlow / Keras
- Google Colab

## 🚀 Conclusion
The model successfully generates text by learning patterns from the dataset using LSTM.
